# lwjgl3ify

**Запуск Minecraft 1.7.10 на LWJGL3 и современной Java (21–25).**

Часть проекта **[Axiom Odyssey](https://github.com/AxiomOdyssey)**.

---

Патчи уровня загрузчика (LWJGL3 + Java-агент), позволяющие Minecraft 1.7.10
запускаться не на устаревшей Java 8, а на актуальной JVM.

Это независимо поддерживаемая копия в рамках проекта **Axiom Odyssey** —
развитие ведётся здесь, на собственной линии, без привязки к другим репозиториям.
