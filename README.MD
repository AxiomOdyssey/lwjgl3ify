---

Патчи уровня загрузчика (**LWJGL3 + Java-агент**), позволяющие Minecraft 1.7.10
запускаться не на устаревшей Java 8, а на актуальной JVM.

Независимо поддерживаемая копия в рамках проекта **Axiom Odyssey** — развитие
ведётся здесь, на собственной линии, без привязки к другим репозиториям.
